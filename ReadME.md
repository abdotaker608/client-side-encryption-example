# Client Side Encryption Example

This repository is an example of impleneting [client side encryption](https://en.wikipedia.org/wiki/Client-side_encryption) by using the [NodeJS Web Crypto API](https://nodejs.org/docs/latest-v19.x/api/webcrypto.html) on the server and [Browser Web Crypto API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Crypto_API) on the client.
